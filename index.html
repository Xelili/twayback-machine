Welcome to the Twayback Machine

The Twayback Machine makes it ever-so-slightly easier to jump back in time to your friends' Twitter feeds from long ago, for fun and trolling.

It was inspired by archive.org's Wayback Machine and the grounbreaking research (and documentation) from @ryandawidjan and @libovness on TweeTrolling.

The Twayback Machine is maintained by Jonathan Howard. This page was generated by GitHub Pages using the Cayman theme by Jason Long.
